Explanation of AJAX

Ajax call retrives the CD data from cd_catlog xml file. When button clicked, JS script creates an XMLHttpRequest, sending a request to server and processes the response. If response sucessful status == 200, the method "displayCDs()" will be triggered, parsing the XML and display the CD data in table form.

REST API - OpenWeather

The OpenWeather API gives weather data for cities around the world. Its responses is in JSON format, making it easy to parse using JS. Developers can retrieve real-time weather, forecasts, & historical data.

Access Requirements: The API requires a free API key for limited access; premium plans are available for extended features.

Pricing: OpenWeather offers both free and paid tiers, with costs depending on the number of API requests made per month.
